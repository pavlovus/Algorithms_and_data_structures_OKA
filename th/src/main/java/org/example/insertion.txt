Ідея:
На кожній ітерації і, ми обмінюємо елемент a[i] місцями з кожним елементом, щo більше за нього з лівої сторони.

Реалізація:
public static void sort(Comparable[] a){
   int n = a.length;
   for (int i=0;i<n;i++){
      for (int j = i;j>0;j--)
         if (less(a[j],a[j-1]))
            exch(a,j,j-1);
         else break;
   }
}

	Складність:

Найкращий
К-сть порівнянь N − 1
К-сть перестановок 0
Оцінка O(N)

Середній
К-сть порівнянь ~ N² / 2
К-сть перестановок ~ N² / 2
O(N²)

Найгірший
К-сть порівнянь N² / 2
К-сть перестановок N² / 2
Оцінка O(N²)

Особливості:
Час виконання чутливий до вхідних даних
Стабільний(стійкий)
