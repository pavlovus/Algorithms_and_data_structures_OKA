Ідея:
на кожній ітерації і, ми знаходимо min мінімальний елемент і міняємо місцями a[i] і a[min]

Реалізація:
public static void sort(Comparable[] a){
   int n = a.length;
   for (int i=0;i<n;i++){
      int min = i;
      for (int j = i+1;j<n;j++)
         if (less(a[j],a[min]))
            min = j;
      exch(a,i,min);
   }
}

Складність:

Будь-який випадок
К-сть порівнянь N² / 2
К-сть перестановок N-1
Оцінка O(N²)

Особливості:
Час виконання не чутливий до вхідних даних
Нестабільний
