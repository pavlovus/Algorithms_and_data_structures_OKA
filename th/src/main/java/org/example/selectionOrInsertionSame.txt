Для масиву з усіма однаковими ключами сортування вставкою працює швидше за сортування вибором,
оскільки воно зробить лише N−1 порівнянь і 0 обмінів (бо на кожному кроці елемент вже на своєму місці),
тоді як сортування вибором завжди знаходить мінімальний елемент у правому підмасиві
та обмінює його з поточною позицією, виконуючи приблизно N^2/2 порівнянь і N обмінів,
незалежно від того, що всі елементи однакові
