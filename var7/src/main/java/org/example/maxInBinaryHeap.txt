При видобуванні максимального елемента з бінарної купи розміром N, що не містить однакових ключів,
мінімальна кількість обмінів дорівнює 1.
Це відбувається у найкращому випадку, коли останній доданий елемент після обміну з коренем (видобутим максимальним)
вже знаходиться на правильному місці і не потребує додаткових перестановок по купі
